環境變數設定

將 sample.env 重新命名成 .env 並修改 MONGODB_URL 與 JWT_SECRET

```
sapmle.env
MONGODB\_URL =(自己的database)
PORT=3001
JWT\_SECRET=(自訂key)
```

在本地端運行

```
npm run dev
```
